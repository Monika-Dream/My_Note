英语作文句型
 - 标准格式 ( 需要背过 )
 
``` 开头的套话
	 - I im delighted to hear you that you want to ...		
		 - 我很高兴听到你想去...

	 - How is it going ? I am so glad to hear you will do sth.
	 	 - 最近怎么样 ? 我很高兴听到你要做...

	 - I`m gald to know that you will do sth. I`d like to give you some suggestions.
	 	 - 很高兴得知你要做 sth 我想要给你一些建议

	 - For sth to choose, I highly recommend sth.
	 	 - 对于你要选择的东西，我高度推荐某个东西.

	 - It is great to hear from you. knowing you want to do sth. I would like to recommend one to you.
	 	 - 收到你的来信很开心, 得知你想要去做 ... 我想给你推荐一个 ...
```
```中间的过度
	 - I heartily recommend sth to you based on the following reasons.
	 	 - 我发自内心的推荐 ____ 给你，基于以下原因
```
```结尾的套话
	 - I hope my introduction is helpful.Please let me know if you have further questions.
	 	 - 希望我的介绍对你有所帮助.如果您有其他问题, 请告诉我.

	 - I really hope my advice will be helpful. wish you a great success.
	 	 - 我真心希望我的建议有用. 希望你成功.

 	 - Does this sth appeal to you? If you are interested, do sth. You will not be disappionted, I pormise!
 	 	 - 这个东西你感兴趣吗？如果要是你感兴趣的话，你可以... 。你不会失望的，我保证。
```

```推荐类文章搭配
	1. - Choose an impressive ( program ) to gain the recognition of the ( judges / audience ).
	 	 - 选择一个令人印象深刻的 ? 来得到 ? 的认可

	2. - Practice your ( performance ) to express yourself more ( clearly ) and ( fluently )
	 	 - 练习你的 ? 来让你自己变得更加 ? 和 ?

	3. - Speaking / acting skills - help maximize the effectiveness of your performance.
	 	 - ? 技能, 能最大化有效性于你的 ?

	   - 1/2/3 三个句型是用于 表演 ( performance ) / 比赛 ( contest ) 这种类似的建议文章中可以使用的
```

 - 通用格式
```

 	 - What takes sth to a new height is sth
 	 	 - 递进句式，引出高潮环节
 	 	 	 - 例子：What took the activity to a new height was the final persentation.
 	 	 		 - 把这个活动带到新的高度的就是最后的这个展示

 	 - If you ask what represents the peak of _____ ,many people would say the _____
 	 	 - 如果你问什么代表着 ____ 的最高峰，许多人会说是 ____
 	 	 	 - 例子：If you ask what represents the peak of Chinese history, many people would say the Terracotta Army.
 	 	 	 	 - 如果你问什么代表着中国历史的最高峰, 许多人会回答兵马俑。

 	 - Search of related background knowledge.
 	 	 - 搜索背景知识
 	 	 - 适用于 [ 活动准备 ]

 	 - Fully understand the conception of sth.
 	 	 - 完全理解某事的概念
 	 	 - 适用于 [ 活动准备 ]

 	 - Does this sth appeal to you? If you are interested, do sth. You will not be disappionted, I pormise!
 	 	 - 这个东西你感兴趣吗？如果要是你感兴趣的话，你可以... 。你不会失望的，我保证。

 	 - I strongly suggest you do sth. for it will do sth.
 	 	 - 我强烈建议你去做... 因为它会有一个...的结果
 	 	 - 建议句式 + 原因，增强说服力

 	 - Have a great chance to do sth.
 	 	 - 有机会去做某事
```

 - 如何写作？
```
名词短语
	 - 核心名词务必要 【 准确 】
	 - 核心名词的前面用形容词修饰 【 情感和档次 】
	 - 核心名词的后面用介词短语，同位语，破折号解释 【 具体化，细节化 】
	 	例子： delicate [ gifts ] for my friends at home.

作文逻辑： 背景 - 特点 - 总结 - 客观特点 - 主观感受 - 期望 - 期望 - 结束

长句写作模式：
	 - 1.介词短语： 时间 / 地点 / 身份 【 As 】 / 性质 【 As 】 / 伴随状态 【 with / without 】
	 	例子： Without the assistance from my teacher and the support from my friends. 
	 			 -  I would never have succeeded

	 - 2.事例列举： such as / for example / like A, B and C
	 - 3.非谓语动词： doing - 同时 / having done 先后 / to do 目的 / done 感受
	 - 4.主从句逻辑关系： what / which / where / when
```
